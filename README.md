# UtaFit MVP

UtaFitは、曲名だけでなく「自分の音域で歌えるか」「歌詞のどこをどんな声で歌うか」から曲を探せる歌唱支援Webアプリです。

今回は添付要件の**Phase 1**を実装しています。アカウント、お気に入り、履歴、Supabase保存、AI候補はPhase 2以降へ分けています。

## できること

- 21曲のカード一覧（実在20曲 + 注釈デモ用の架空曲1曲）
- 曲名、アーティスト、特徴、雰囲気の入力中検索
- 音域、難易度、男女カテゴリ、テンポ、歌唱指標の絞り込み
- 7種類の並び替え
- 検索条件をURLへ保存
- 曲詳細、音域幅、歌唱指標、似ている曲
- 架空歌詞への歌唱方法表示とクリック解説
- 重なる注釈の二重線表示
- セクション切り替え、表示情報の切り替え
- 1行ずつ進める練習モード（進捗はこの端末に保存）
- 音域と好みからの匿名おすすめ診断
- ローカル開発専用の簡易注釈編集画面
- 404、読み込み中、検索0件の画面
- 計算ロジックの自動テスト

## 大切なデータ上の注意

実在曲の音域・BPM・難易度は、画面の動作確認用のサンプル値であり、正確な分析値として掲載していません。実在曲の歌詞は保存していません。歌詞と注釈は、このMVP専用の架空曲「夜明けのキャンバス」だけで確認できます。

## 起動方法

Macのターミナルで、このフォルダを開いて次の順に実行します。

```bash
pnpm install
```

```bash
pnpm dev
```

ブラウザで `http://localhost:3000` を開きます。pnpmがない場合は、`npm install`、`npm run dev`でも起動できます。Node.js 20.9以上が必要です。

## 主な画面

- `/`：サービス説明、検索、おすすめ曲
- `/songs`：曲一覧、検索、絞り込み、並び替え
- `/songs/[id]`：曲詳細と歌い方
- `/songs/[id]/practice`：1行練習
- `/recommend`：音域から匿名診断
- `/project-story`：制作意図、設計、学びをまとめたブログ記事
- `/admin`：管理者専用案内

ローカル開発時だけ、`/admin?preview=1`で簡易注釈編集を確認できます。本番環境ではこの方法で開けません。

## フォルダ構成

```text
app/                    画面とURL
components/             再利用する画面部品
data/                   曲、架空歌詞、技法のサンプルデータ
lib/                    音名、検索、注釈、おすすめ計算
types/                  TypeScriptのデータ型
tests/                  計算ロジックの自動テスト
docs/                   次のPhaseの設計メモ
```

画面と計算処理を分けているため、将来`data/`をSupabaseの取得処理へ置き換えやすい構成です。

## 現在の制限

- 実在曲の歌唱数値はサンプルで、検証済みデータではありません。
- 実在曲の歌詞、音源再生、音源同期はありません。
- 注釈編集の保存はローカルプレビューだけです。
- アカウント、お気に入り、履歴、個人データ保存はまだありません。
- おすすめは機械学習ではなく、独立したルールベース関数です。

## 次のPhase

Phase 2ではSupabase Auth、Database、Row Level Securityを追加し、お気に入り、歌いたい曲、練習中、閲覧・検索履歴、正式な歌詞検索を実装します。設計方針は `docs/phase2-supabase-design.md` にまとめています。
